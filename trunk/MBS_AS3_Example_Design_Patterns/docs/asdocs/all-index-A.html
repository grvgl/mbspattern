AbstractTemplate — class, package com.shivolve.mbs.as3.mind.templatemethod
Template Method defines steps of an algorithm. Defers the exact steps of the algorithm to a subclass. Template Method pattern creates a template for an algorithm. Template Method is an important technique for code reuse. Template Method has an abstract class with abstract methods, concrete methods and hooks. Template Method encapsulates algorithm using inheritance (whereas Strategy Pattern encapsulates algorithm using composition).
